## Локальний запуск проекту

1. Клонуйте репозиторій:
   git clone https://github.com

2. Перейдіть у папку проекту:
   cd название-папки

3. Встановіть залежності:
   npm install

4. Запустіть проект у режимі розробки:
   npm run dev

Застосунок буде доступний за адресою http://localhost:5173

## Опис функціоналу

SPA застосунок для розшифровки VIN-коду автомобіля з використанням API NHTSA.

**Основні можливості:**
- Пошук інформації про автомобіль за VIN-кодом.
- Збереження історії 3-х останніх запитів (з використанням LocalStorage).
- Перегляд списку всіх доступних змінних автомобіля (/variables).
- Детальна інформація про кожну змінну за її ID (/variables/{id}).
- Адаптивна верстка (від 420px до 1440px) на чистому CSS без використання фреймворків.
